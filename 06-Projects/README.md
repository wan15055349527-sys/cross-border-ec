# 📦 Project Status

> 项目管理中心 — 跟踪所有活跃项目。

## 标签规范

- `status/active` — 进行中
- `status/paused` — 暂停
- `status/planning` — 规划中
- `status/done` — 已完成

## 项目结构规范

每个项目在 `06-Projects/` 下创建一个文件夹，包含:

```
06-Projects/{Project-Name}/
├── Project-Status.md    # 项目状态（必需）
├── ...                  # 项目相关文件
```
