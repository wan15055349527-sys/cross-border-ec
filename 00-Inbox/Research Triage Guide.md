# 🔄 Research Triage Guide

## 处理流程

`
Downloaded ──→ 需要清洗? ──→ Yes ──→ Research Cleaner ──→ Cleaned
                │                      │
                No                     No
                │                      │
                ▼                      ▼
        直接分类 ──────────────→ 04-Research/{领域}/
                                   │
                                   ▼
                             更新 Topic Hub
`

## triage 决策树

| 条件 | 操作 |
|------|------|
| 完整高质量文章 | 清洗后分类 |
| 简短笔记/灵感 | 直接移至 97-AI-Memory/Ideas |
| 含可复用工作流 | 清洗后移入 97-AI-Memory/Workflows |
| 商业/策略洞察 | 清洗后移入 97-AI-Memory/Knowledge Summary |
| 已读不需要保留 | 移入 99-Archive |
